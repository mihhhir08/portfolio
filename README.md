# mihirsinhchavda.com

Personal site. Software engineer, I ship AI products end to end.

Zero-build static site: one HTML file, one stylesheet, one script. No framework, no bundler, no dependencies.

## Details

- WebGL2 dithered noise shader background (Perlin fbm + 8×8 Bayer matrix), theme-aware, pauses off-screen
- Command palette (`⌘K`) and a terminal easter egg (press `` ` ``)
- Pixel-sprite cat that follows the cursor, drawn on a 2D canvas from string-array frames
- Live visitor counter with localStorage guard and deterministic fallback
- Dark/light theme, `prefers-reduced-motion` respected, semantic HTML + JSON-LD

## Run

Open `index.html`. That's it.
